# Melbourne House Price Predictor
A machine learning model which predicts the price of a house in Melbourne based upon features like location, number of rooms etc.

Model Accuracy is 82.03%

Trained in scikit-learn using Random Forest Algorithm. Built backend in Django and frontend in Vue.js.

This project was inspired from the course on Kaggle - Intro to Machine Learning. Link: https://www.kaggle.com/learn/intro-to-machine-learning.
The dataset was also a part of this course.
